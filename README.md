## 📖 About This Project
This repository contains the front-end architecture for the official HaiDev Agency digital storefront. It is built entirely on a lightweight, high-performance Vanilla web stack, demonstrating a deep commitment to rigorous technical web standards, accessibility, and search engine discoverability.

## ⚡ Technical Highlights & SEO Architecture
This site was engineered with a strict focus on Technical SEO and performance optimization, moving beyond just UI design to include professional-grade web standards:
* **Search Engine Optimization:** Includes highly optimized `sitemap.xml`, `robots.txt`, and verified Google Search Console integration for maximum indexing efficiency.
* **Dynamic Data Handling:** Utilizes a `projects.json` architecture to dynamically manage and render portfolio content via Vanilla JavaScript.
* **Multi-Page Architecture:** Cleanly separated modular HTML files (`about.html`, `services.html`, `portfolio.html`, etc.) for semantic structure and clear user navigation.

## 🛠️ Tech Stack
* **Markup & Styling:** HTML5, CSS3
* **Scripting & Data:** Vanilla JavaScript, JSON
* **Deployment & Version Control:** Git, GitHub Pages
